什麼是使用者體驗（UX）？

核心定義

使用者體驗是使用者在與產品、系統或生態系統的其他部分互動之前、期間和之後所感受到的所有效應的總和。

UX的關鍵特徵：

  • 源於互動 - 由使用者與產品的互動產生
  • 效應的總和 - 包含完整的感知、解釋和情感
  • 內在感受 - 由使用者主觀體驗
  • 情境相關 - 受使用環境和生態系統影響
UX不僅是產品功能，更是使用者的完整體驗旅程

運算與互動的演進

技術發展歷程

1970年代末

個人電腦出現，主要是資料欄位顯示

1980-1990年代

圖形使用者介面（GUI）、網際網路興起

2000年代至今

行動裝置、穿戴式裝置、物聯網普及

互動模式的擴展

  • 從WIMP到多模態 - 視窗、圖示、選單、指標 → 觸控、語音、手勢
  • 消失的技術 - 融入生活環境，變得無形
  • 普及運算 - 智慧手機、手錶、汽車、家電

活動導向互動

什麼是活動導向互動？

一組多重、重疊和相關的任務線程，都服務於使用者生態系統內的共同工作目標。

三種層次的活動導向互動：

  • 單一裝置 - 在一個裝置上完成複合任務
  • 跨多個裝置 - Apple Watch + iPhone 的生態系統
  • 跨系統生態 - 涉及多方、多裝置、多位置

實例：電線掉落修復

使用者發現停電 → 聯絡鄰居確認 → 致電電力公司 → 客服建立工單 → 技師接收任務 → 現場修復 → 回報完成

整個流程跨越多個系統、裝置和人員，但對使用者來說是一個完整活動。

使用者體驗的四大組成要素

🔧 易用性 (Usability)

易於使用、高效率、低錯誤率、易學習、易記憶

⚡ 有用性 (Usefulness)

功能強大、滿足需求、解決問題的能力

❤️ 情感影響 (Emotional Impact)

愉悅感、滿意度、美感體驗、樂趣

🎯 意義性 (Meaningfulness)

與使用者生活的長期連結、個人價值

優秀的UX設計需要平衡所有四個要素

情感影響的重要性

為什麼情感很重要？

"使用者不再滿足於僅僅是易用性的效率和有效性；他們也在尋求情感滿足。"

"有吸引力的東西讓人感覺良好。" - 唐·諾曼

情感影響的表現形式：

  • 使用的喜悅、愉悅感、興奮
  • 美學體驗、新穎性、驚喜
  • 身分認同、社會地位
  • 擁有的驕傲、產品親和力

iPad 成功案例

iPad 被批評為"多餘"，但成為史上最成功的個人電子裝置。原因不在於功能理性，而在於情感誘惑 - 持有精美科技藝術品的體驗感受。

UX重要性的演進

早期階段

容忍不良設計，技術神秘感，"跳舞熊"軟體

1990年代

易用性工程興起，關注評估方法

2000年代至今

UX設計成為核心，超越易用性到全方位體驗

現今重要性

"商業策略和使用者體驗設計之間不再有任何真正的區別" - IBM 高級副總裁

"我們生活在使用者體驗時代" - 《大西洋》雜誌

不良UX設計的代價

高成本實例：

🚗 汽車安全

不良車載系統UX導致駕駛分心，造成交通事故、傷亡

✈️ 航空安全

1999年埃及航空990號航班墜毀，原因：駕駛艙控制設計易用性不佳

🚢 海事安全

USS McCain 碰撞事故，歸因於導航控制台UX設計問題

🏥 醫療安全

自動化醫院系統的22種錯誤用藥方式，多數為經典易用性問題

良好的UX設計不只是錦上添花，而是攸關安全的必需品

服務體驗設計

服務設計定義

在"交易旅程"內為使用者或客戶體驗而進行的UX設計，關注客戶接觸點。

手術服務旅程範例

  1. 初診 - 家庭醫生檢查症狀
  2. 檢測 - 診所進行影像檢查
  3. 入院 - 醫院導航、報到、等候
  4. 手術 - 手術執行
  5. 術後 - 處方、追蹤、復健

每個環節都是潛在的UX設計點

服務設計特點：

  • 跨時間、跨地點的體驗
  • 多個接觸點整合
  • 考慮邊緣情況和問題點

UX的商業價值

從成本正當化到策略核心

過去： 需要證明UX投資回報率

現在： UX成為商業策略不可分割的部分

UX價值體現：

  • 降低支援成本 - 好設計減少客服需求
  • 提升客戶滿意度 - 增加忠誠度和回購
  • 品牌差異化 - 在競爭中脫穎而出
  • 減少開發成本 - 一次做對，避免重工
  • 縮短產品上市時間
"設計思維"的普及讓企業開始像設計師一樣思考

蘋果效應

蘋果的成功證明了優秀UX設計的商業價值，引發全行業對UX的重視。

HCI與UX的歷史根源

1940-1950年代

人因工程興起，費茨、瓊斯、查帕尼斯等先驅

1960年代

恩格爾巴特發明滑鼠，電腦圖形學起步

1980年代

GUI發展，施樂之星、蘋果Lisa和Macintosh

1983年

首屆CHI會議，HCI學科正式成形

2000年代

從易用性擴展到全面使用者體驗

多學科整合

HCI與UX結合了心理學、認知科學、工業設計、電腦科學等多個領域的知識。

關鍵要點總結

🎯 核心概念

  • UX是使用者與產品互動的完整感受總和
  • 包含互動前、中、後的所有效應
  • 受情境和生態系統影響

🔧 四大組成要素

  • 易用性 - 基礎功能表現
  • 有用性 - 解決問題能力
  • 情感影響 - 愉悅和滿足感
  • 意義性 - 長期價值連結

📈 戰略重要性

  • 從技術補充到商業核心
  • 攸關安全和成本效益
  • 決定產品成功的關鍵因素
優秀的UX設計是現代產品開發的必需品，不是可選項
